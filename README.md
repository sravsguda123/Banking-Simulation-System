# 🚀 Java Banking Simulation System

A complete **Core Java OOP-based Banking System** with a built-in REST API and a modern interactive web dashboard.

---

## ✨ Features

* 🧠 **Object-Oriented Design**

  * Inheritance (`Account`, `SavingsAccount`)
  * Encapsulation (secure balance handling)
  * Polymorphism (dynamic behavior)

* 📊 **Collections Framework**

  * Uses `HashMap` and `ArrayList` as an in-memory database
  * No external setup required

* 🌐 **REST API (Backend)**

  * Built using Java’s `com.sun.net.httpserver`
  * Handles transactions like deposit, withdraw, transfer

* 💻 **Modern Web UI (Frontend)**

  * Clean fintech-style dashboard
  * Built with HTML, CSS, JavaScript
  * Real-time interaction with backend

---

## 🛠️ Tech Stack

* Java (Core + OOP)
* HTML, CSS, JavaScript
* REST API

---

## ▶️ How to Run

1. Run the backend:

   * Double-click `run.bat`
     *(or compile manually: `javac Main.java` and run `java Main`)*

2. Start the application:

   * Server runs on **http://localhost:8080**

3. Open frontend:

   * Go to `frontend/index.html` in browser

4. Test features:

   * Deposit 💰
   * Withdraw 💸
   * Transfer 🔄
