# Java Banking Simulation System

A complete Core Java object-oriented banking simulation featuring a built-in REST API and a stunning modern web dashboard.

## Features
- **Core OOP Concepts:** Heavily utilizes Inheritance (`Account`, `SavingsAccount`), Encapsulation (protected balances), and Polymorphism.
- **Collections Framework:** In-memory database built using Java `HashMap` and `ArrayList` ensuring zero configuration required to test.
- **REST API:** Utilizes Java's built-in `com.sun.net.httpserver` to expose endpoints.
- **Web UI:** A dynamic, beautiful "Fintech" dashboard built with Vanilla Javascript and CSS.

## How to Run

1. Double-click the `run.bat` file in this folder.
2. The batch script will compile the Java code and start the server on port 8080.
3. Open the `frontend/index.html` file in any web browser.
4. Perform deposits, withdrawals, and transfers to test the OOP logic!

## Recording your demo
The UI looks premium. Just start the server, open the HTML file, and record your screen as you interact with the dashboard!
